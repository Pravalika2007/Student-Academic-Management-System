# StudentAcademicSystem

This Python project is a **Student Academic Record Management System** using **Object-Oriented Programming (OOP)**.  
It allows a college to manage student academic records, calculate CGPA, track fee balance, and display student details.

### Features:

- Store college details  
- Register multiple students  
- Calculate average marks  
- Calculate CGPA  
- Track fee payments and balance  
- Display complete student academic report  

---

## 📂 Repository Files

- **`student.py`** – The main Python program containing `Student` and `College` classes with all logic.  
- **`output1.txt`** – Sample output of the program.  
- **`Output (1).png`** – Screenshot of the program execution.
